# Cloud Service App

Aplicación web que simula un servicio en la nube con registro y autenticación de usuarios.

## Tecnologías
- HTML, CSS, JavaScript
- Node.js
- Express

## Funcionalidades
- Registro de usuarios
- Inicio de sesión
- Dashboard
- Consumo de API con fetch

## Arquitectura
El sistema sigue una arquitectura cliente-servidor:
- Frontend: Interfaz de usuario
- Backend: API REST
- Datos: Almacenamiento en JSON
